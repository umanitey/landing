Installation sur Android

Merci de votre intérêt pour ElMoslim ! Notre application n'est pas encore disponible sur le Google Play Store, mais vous pouvez l'installer directement via notre fichier APK sécurisé.

En bas de page, une vidéo explicative est disponible pour vous guider dans l'installation.

Note importante : L'installation d'applications en dehors du Play Store nécessite quelques étapes supplémentaires pour des raisons de sécurité. Ces étapes sont standards sur Android et protègent votre appareil.

Procédure d'installation

1. Téléchargez le fichier APK en cliquant sur le bouton ci-dessous :

Télécharger

2. Lorsque le téléchargement commence, votre navigateur pourrait vous avertir que "ce type de fichier peut être dangereux". C'est un message standard pour tous les APK. Cliquez sur "Télécharger quand même".

3. Une fois le téléchargement terminé, ouvrez le fichier APK.

4. Si c'est la première fois que vous installez une application hors Play Store, Android vous demandera d'autoriser l'installation depuis cette source :

  1. Dans la fenêtre qui s'affiche, appuyez sur "Paramètres"
  2. Activez l'option "Autoriser depuis cette source"
  3. Revenez à l'écran précédent (bouton retour)

5. Appuyez sur "Installer" et attendez la fin de l'installation

6. Félicitations ! Vous avez installé ElMoslim avec succès.

Questions fréquentes

Est-ce sécurisé d'installer cette application ?

Oui, notre APK est signé numériquement par nos soins et ne contient aucun code malveillant. L'avertissement que vous voyez est standard pour toute application installée en dehors du Play Store.

Pourquoi l'application n'est pas sur le Play Store ?

Notre application est actuellement en phase de validation finale pour le Google Play Store. Plutôt que de retarder son lancement, nous avons choisi de la rendre disponible directement via APK pour vous permettre d'en profiter dès maintenant.

Comment recevoir les mises à jour ?

Pour le moment, vous devrez revisiter cette page pour télécharger les nouvelles versions. Une fois disponible sur le Play Store, vous pourrez bénéficier des mises à jour automatiques.

J'ai un problème avec l'installation, que faire ?

Si vous rencontrez des difficultés, n'hésitez pas à nous contacter via notre page de contact ou nos réseaux sociaux.
